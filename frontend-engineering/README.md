# Frontend Engineering Skill

Countermeasure for LLM frontend structure failures: wrong stack, vanilla monoliths, dual systems.

Install: copy or clone this folder to `~/.cursor/skills/frontend-engineering/`.

Charter: `../_suite-charters/frontend-engineering.md`

```bash
node scripts/profile-stack.js --root .
node scripts/check-structure.js --root . --strict
```
